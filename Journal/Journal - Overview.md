# Journal — Overview

> Journal ใช้เก็บร่องรอยของความคิด, decision, progress, และ next step

## Entry Points

- [[Daily/2026-04-11|Today's Note]]
- [[Templates/Daily Note Template|Daily Note Template]]
- [[Templates/Weekly Review Template|Weekly Review Template]]
- [[Concepts/Obsidian Node Growth Workflow|Node Growth Workflow]]

## When To Write

- ตอนมี decision สำคัญ
- ตอนแก้ปัญหาแล้วได้บทเรียน
- ตอนมี next step ที่ไม่อยากให้หาย
- ตอนต้องการทบทวนว่าสัปดาห์นี้กำลังพลังไปทางไหน

## When Not To Write

- คำถามสั้นที่ไม่มีผลต่ออะไรต่อ
- งานจุกจิกที่ไม่มี insight
- session ที่ไม่ได้สร้าง decision, artifact, หรือ action

## Suggested Rhythm

- Daily note: เขียนเฉพาะวันที่มีสาระพอให้ทิ้งร่องรอย
- Weekly review: สรุป win, open loop, และโฟกัสของสัปดาห์ถัดไป

> [!tip]
> Daily note ไม่ใช่ transcript ทั้งหมดของการคุยกับ AI
> เก็บแค่สิ่งที่ควรย้อนกลับมาอ่าน
