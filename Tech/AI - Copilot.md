# Copilot — GitHub Copilot Education Plan

> เรียกด้วย: **`Copilot`**
> Provider: [[Provider - GitHub Microsoft|GitHub / Microsoft]] + [[Provider - OpenAI|OpenAI]]

---

## ข้อมูลทั่วไป

- · Provider: GitHub / Microsoft / OpenAI
- · Plan: Education Plan
- · Interface หลัก: VS Code, JetBrains, GitHub.com, Copilot CLI

---

## จุดแข็ง

- · Inline autocomplete — แนะนำโค้ดขณะพิมพ์ real-time
- · Context-aware — อ่าน codebase ใน repo ปัจจุบัน
- · GitHub integration — ช่วย review PR, explain commit บน GitHub ได้
- · Copilot Chat — ถามคำถามเกี่ยวกับโค้ดใน sidebar
- · Multi-file edit — แก้หลายไฟล์พร้อมกันใน VS Code
- · Copilot CLI — อธิบาย shell command, suggest git command
- · ฟรีสำหรับ Education plan (verified student)

---

## เมื่อไหร่ควรใช้

- · เขียนโค้ดปกติทุกวัน — ให้มัน autocomplete
- · เปิด PR แล้วให้ review code บน GitHub
- · ถามคำถามเร็ว ๆ เกี่ยวกับ function หรือ class ที่กำลัง edit
- · หา shell command ที่ลืม

---

## วิธีเรียกใช้

- · VS Code: install GitHub Copilot extension → เปิดอัตโนมัติ
- · Chat: `Ctrl+Shift+I` (VS Code) หรือ icon ใน sidebar
- · CLI: `gh copilot suggest`, `gh copilot explain`
- · GitHub.com: Copilot icon ใน PR/Issues
