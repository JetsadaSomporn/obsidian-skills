# Kilo Code — Agentic Coding Agent

> เรียกด้วย: **`Kilo`**
> Provider: [[Provider - Kilo Code|Kilo Code]]

---

## ข้อมูลทั่วไป

- · Provider: Kilo Code
- · Interface หลัก: VS Code Extension

---

## จุดแข็ง

- · Agentic coding — คล้าย Claude Code / Cline แต่ model-agnostic
- · ต่อได้กับหลาย model — Claude, GPT, Gemini, local models
- · ใช้ model ของตัวเองได้ (Bring Your Own Key)
- · Multi-file editing — แก้ไฟล์หลายตัวพร้อมกัน
- · Terminal access — รัน command ได้จากใน VS Code
- · Browser tool — เปิด browser ดู UI ได้
- · Cost tracking — แสดง token ที่ใช้และค่าใช้จ่ายได้

---

## เมื่อไหร่ควรใช้

- · ต้องการ agentic coding แต่อยากเลือก model เองได้
- · ใช้ local model (Ollama) เพื่อประหยัดค่าใช้จ่าย
- · งาน coding ที่ต้องการ multi-file edit บน VS Code

---

## วิธีเรียกใช้

- · VS Code: install Kilo Code extension
- · เปิด sidebar → เลือก model → type prompt
