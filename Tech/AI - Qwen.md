# Qwen — Free Plan

> เรียกด้วย: **`qwen`**
> Provider: [[Provider - Alibaba Cloud|Alibaba Cloud]] / Tongyi

---

## ข้อมูลทั่วไป

- · Provider: Alibaba Cloud / Tongyi
- · Plan: Free Plan
- · Interface หลัก: qwen.aliyun.com, Qwen API

---

## จุดแข็ง

- · ภาษาไทยและภาษาจีน — เข้าใจ context ได้ดีกว่าตัวอื่น
- · Multimodal — อ่านรูปได้
- · Long context — Qwen รุ่นใหม่รองรับ context ยาวมาก
- · ฟรี — ใช้ได้โดยไม่มีค่าใช้จ่าย
- · Open source versions — มี Qwen-7B, Qwen-14B ที่ run local ได้
- · Code generation ภาษาไทย — อธิบายและเขียนโค้ดเป็นภาษาไทยได้
- · Qwen2.5-Coder — รุ่นเฉพาะทาง coding

---

## เมื่อไหร่ควรใช้

- · ต้องการคำอธิบายภาษาไทยที่ชัดเจน
- · งาน content ภาษาไทยหรือภาษาจีน
- · ต้องการ AI ฟรีสำหรับ task ทั่วไป
- · ทดลอง run model local (open source version)
