# Obsidian + AI Workflow

> Graph ที่มีค่าคือ graph ที่สะท้อนสิ่งที่ **สมองเราเชื่อม** ไม่ใช่สิ่งที่ AI คิดให้

## หลักการหลัก

ใช้ AI เป็น **rubber duck ที่ฉลาดกว่า** — คุยเพื่อ process input แต่ **เราเป็นคนเขียน note เอง**

| ให้ AI ทำ | เราทำเอง |
|-----------|----------|
| Summarize บทความ | เขียน note ด้วยภาษาตัวเอง |
| Extract key ideas | ตัดสินว่า idea ไหนสำคัญกับเรา |
| ถามคำถามเพื่อ clarify | พิมพ์ wikilink ด้วยมือ |
| Debug หรือ explain concept | เชื่อม concept ในหัว |

## สัญญาณเตือนว่าทำผิด

- Note มี structure สวยมากแต่ไม่มี insight ของเราอยู่เลย
- Wikilink ถูกสร้างโดย AI ไม่ใช่เราหยุดคิดเอง
- Vault optimize ให้ AI อ่าน ไม่ใช่ให้สมองเราเชื่อม

## Workflow ที่ถูก

```
เจอ idea / ทำงานเสร็จ
     ↓
คุยกับ Claude เพื่อ distill (rubber duck)
     ↓
เราหยุดคิด: "นี่เชื่อมกับอะไรที่เคยรู้?"
     ↓
เขียน note ด้วยภาษาตัวเอง + พิมพ์ wikilink เอง
```

> [!tip] Wikilink moment
> ทุกครั้งที่พิมพ์ `[[]]` คือ moment ที่มีค่าที่สุด — นั่นคือตอนที่สมองเชื่อม concept จริงๆ อย่าให้ AI ข้ามขั้นตอนนี้แทน
